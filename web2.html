Web 2.0

The Social Web — 2004 to Present

What is Web 2.0?

Web 2.0 is the second generation of the Web, emerging around 2004 and continuing to the present day. It transformed the Web from a read-only medium into a read-write platform where users became active content creators rather than passive consumers.

Web 2.0 brought social media, blogs, wikis, video sharing, and e-commerce. Dynamic content, user accounts, APIs, and AJAX technologies define this era.

Web 2.0 gave birth to Facebook, YouTube, Twitter, Wikipedia, Google Maps, and the entire modern social internet.

Key Characteristics

  • User-generated content — anyone can publish, comment, share
  • Social networking platforms (Facebook, Twitter, Instagram)
  • Dynamic pages — content updates without page reload (AJAX)
  • Rich web applications (Google Docs, Gmail)
  • APIs allowing services to communicate with each other
  • Mobile web and responsive design
  • Cloud storage and Software as a Service (SaaS)
