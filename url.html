Uniform Resource Locator

URL — The address of every resource on the Web

What is a URL?

A URL (Uniform Resource Locator) is the unique address used to locate a specific resource on the Internet. Every web page, image, video, file, or API endpoint has its own URL. It tells the browser what protocol to use, which server to contact, and exactly which resource to retrieve.

Every time you click a link or type an address into your browser, you are using a URL.

Parts of a URL

Example: https://www.strathmore.edu/admissions?year=2026#form
