Domain Name Server

DNS — The Internet'\''s Phone Book

What is DNS?

The Domain Name System (DNS) is the Internet's phone book. Computers communicate using numerical IP addresses (e.g. 142.250.80.46) but humans use domain names (e.g. google.com). DNS automatically translates domain names into IP addresses so browsers can find the correct server.

Without DNS you would have to memorise the IP address of every website you want to visit!

DNS Resolution Process

  • Step 1: Browser checks its local cache — if the IP is already saved, it skips the rest
  • Step 2: Asks the DNS Resolver (your ISP's middleman)
  • Step 3: Resolver asks a Root DNS Server — directs to the correct TLD server
  • Step 4: TLD server (e.g. .com, .edu) directs to the Authoritative DNS server
  • Step 5: Authoritative server returns the actual IP address
  • Step 6: Browser connects to the server using that IP
