Intranet

Private Network — Internal organisational network

What is an Intranet?

An Intranet is a private network that uses the same technologies as the Internet (HTTP, TCP/IP, HTML) but is restricted exclusively to the employees of a specific organisation. It is completely inaccessible to the general public.

Intranets are used for internal communication, document sharing, HR portals, announcements, and company databases. Example: Strathmore University's internal student and staff portal.

Same technology as the Internet — but locked behind a firewall so only authorised users inside the organisation can access it.
