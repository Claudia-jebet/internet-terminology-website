Transmission Control Protocol / Internet Protocol

TCP/IP — The backbone of Internet communication

What is TCP/IP?

TCP/IP is the foundational set of communication protocols (rules) that govern how data is sent, received, and routed across the Internet. It is not a single protocol but a suite of protocols working together.

  • IP (Internet Protocol) — handles addressing and routing. Every device on the Internet has a unique IP address. IP breaks data into packets and routes each one to the correct destination.
  • TCP (Transmission Control Protocol) — ensures reliability. It breaks data into packets, sends them, checks they all arrived, and reassembles them in the correct order. If any packet is lost, TCP requests it again.
Think of IP as the postal address system and TCP as the delivery confirmation — making sure every package actually arrives safely.
