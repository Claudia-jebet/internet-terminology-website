Hypertext Markup Language

HTML — The language of the Web

What is HTML?

HTML (Hypertext Markup Language) is the standard language used to create and structure web pages. It uses tags (elements) wrapped in angle brackets to define the structure and meaning of content — headings, paragraphs, links, images, forms, tables, and more.

HTML is NOT a programming language — it has no logic, conditions, or loops. It works together with CSS (styling) and JavaScript (behaviour/interactivity) to build complete web pages.

HTML = Structure. CSS = Style. JavaScript = Behaviour. All three together make a modern web page.
