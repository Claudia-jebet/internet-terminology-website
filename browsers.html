Web Browsers

Client Software — Chrome, Firefox, Safari, Edge

What is a Web Browser?

A web browser is a software application that retrieves, interprets, and displays web pages from the Internet. It acts as the client in the client-server model — it makes requests and receives responses.

Popular browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari, and Opera.

A browser does not just display text — it interprets HTML structure, applies CSS styling, and executes JavaScript to create the interactive pages you see.

How a Browser Works

  • User types a URL and hits Enter
  • Browser performs a DNS lookup to find the server IP address
  • Browser sends an HTTP GET request to the web server
  • Server responds with HTML, CSS, and JavaScript files
  • Browser parses and renders the page visually
