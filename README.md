# Internet Terminology Reference Website

A multi-page website covering 16 core Internet and Web Technology terms.

## Terms Covered
1. World Wide Web (WWW)
2. TCP/IP
3. Web Browsers
4. Web Servers
5. URL
6. DNS
7. HTTP/HTTPS
8. Intranet
9. Extranet
10. Multitier Architecture
11. FTP
12. HTML
13. Web 1.0
14. Web 2.0
15. Web 3.0
16. Web 4.0

## Project Structure
```
internet-terminology-website/
├── index.html          # Home page with all 16 terms listed
├── style.css           # Shared stylesheet for all pages
├── README.md           # Project documentation
├── images/             # All images stored here
│   ├── www.jpg
│   ├── tcpip.jpg
│   └── ...
└── pages/              # Individual term pages
    ├── www.html
    ├── tcpip.html
    ├── browsers.html
    └── ...
```

## How to Run
1. Clone the repository: `git clone <your-repo-url>`
2. Open `index.html` in any web browser
3. Navigate through all 16 terms using the top navigation bar

## Features
- 16 individual HTML pages — one per term
- Fixed navigation bar on every page linking to all terms
- SVG diagrams illustrating each concept
- Images folder with relevant visuals
- Consistent design across all pages
- Fully interlinked pages for seamless navigation
