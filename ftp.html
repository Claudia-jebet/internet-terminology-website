File Transfer Protocol

FTP — Transferring files between client and server

What is FTP?

File Transfer Protocol (FTP) is a standard network protocol used to transfer files between a client and a server over a TCP/IP network. It operates on port 21 and is commonly used by web developers to upload website files to a hosting server.

SFTP (Secure FTP) is the encrypted version that protects file transfers from interception. Common FTP clients include FileZilla and WinSCP.

Unlike HTTP which is used for viewing web pages, FTP is specifically designed for uploading and downloading files between computers.

FTP vs HTTP

  • FTP — designed for transferring files (upload/download). Uses port 21.
  • HTTP — designed for viewing web pages. Uses port 80/443.
  • SFTP — encrypted FTP. Uses port 22.
