Hypertext Transfer Protocol

HTTP / HTTPS — The foundation of Web communication

What is HTTP?

HTTP is the protocol that defines how messages are formatted and transferred between a web browser and a web server. It is the foundation of all data communication on the World Wide Web.

HTTPS is the secure version — it encrypts the connection using SSL/TLS to protect sensitive data like passwords and payment details from being intercepted.

HTTP Methods: GET (retrieve a page), POST (send data to server), PUT (update data), DELETE (remove data)

HTTP Request-Response Cycle

  • 1. User enters URL in browser
  • 2. DNS resolves domain to IP address
  • 3. TCP connection established (3-way handshake)
  • 4. Browser sends HTTP GET request
  • 5. Server processes and finds the resource
  • 6. Server sends HTTP response (200 OK + content)
  • 7. Browser renders the page
