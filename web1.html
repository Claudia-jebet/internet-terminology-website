Web 1.0

The Static Web — 1991 to 2004

What is Web 1.0?

Web 1.0 is the first generation of the World Wide Web, spanning from approximately 1991 to 2004. It is characterised by static HTML pages that users could only read — there was no interactivity, no user accounts, no comments, and no dynamic content.

Web 1.0 is often called the "read-only web" or the "brochure web" because websites functioned like digital brochures — information was published by companies and consumed passively by users.

In Web 1.0, content creators were a tiny minority. The vast majority of users were purely consumers of content with no ability to contribute.

Key Characteristics

  • Static HTML pages — content never changed unless manually edited by the developer
  • No user interaction or user-generated content
  • No logins, no accounts, no personalisation
  • Simple hyperlinks to navigate between pages
  • Slow dial-up connections dominated
  • Examples: Early company websites, basic government pages, online encyclopedias
