World Wide Web

WWW — Invented by Tim Berners-Lee, 1989

What is the World Wide Web?

The World Wide Web is a system of interlinked hypertext documents and multimedia content accessed via the Internet using a web browser. It was invented by Tim Berners-Lee in 1989 while working at CERN.

The Web uses HTTP to transfer HTML pages between servers and browsers. It is the most widely used service on the Internet and forms the foundation of modern online communication.

The WWW is NOT the Internet. The Internet is the physical infrastructure — cables, routers, and servers. The WWW is a service that runs on top of the Internet.

How It Works

The Web is built on three core technologies:

  • HTML — structures the content of web pages
  • HTTP — the protocol used to transfer pages between server and browser
  • URL — the unique address used to locate each resource
